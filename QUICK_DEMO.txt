📱 TRY DIFFERENT ROLES:

1. PATIENT MODE (Booking)
   ├─ Click "Book Now" on landing page
   ├─ Step 1: Select department (Laboratory/Imaging/Consultation)
   ├─ Step 2: Pick date and time
   ├─ Step 3: Enter name, phone, optional problem description
   ├─ Confirmation: See booking ID, print receipt
   └─ Done!

2. ADMIN MODE (Queue Management)
   ├─ Look for admin access point (usually in navbar)
   ├─ Login with any email, select "Admin" role
   ├─ See live queue of pending appointments
   ├─ Click "Check In" to complete appointment
   ├─ Click "Cancel" to cancel appointment
   ├─ View "History" tab for completed appointments
   └─ Done!

3. DOCTOR MODE (Review Appointments)
   ├─ Login with any email, select "Doctor" role
   ├─ See list of appointments on left
   ├─ Click appointment to see full details
   ├─ Review patient info and problem description
   ├─ See home address if applicable
   ├─ Ready to add notes or reschedule (UI ready)
   └─ Done!
